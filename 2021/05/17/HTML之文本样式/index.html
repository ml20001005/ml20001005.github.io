HTML之文本样式

1.文本颜色

rgb的三个属性值分别是: 红 绿 蓝, 数值范围是: 0 ~ 255

rgba的四个属性值分别是: 红 绿 蓝 透明度, rgb数值范围是: 0 ~ 255, a的数值范围是: 0 ~ 1

2.文本对齐方式

左右对齐

3.首行缩进

2em代表两个汉字高度, 汉字为方块字, 所以高度等于宽度, 也就是缩进两个字的宽度

4.行高

当文本内容较少时可以做到文本上下居中

5.装饰

例如去除a标签的默认下划线装饰

6.文本和图片垂直对齐

垂直对齐方式, 可以使图片和文本垂直对齐
