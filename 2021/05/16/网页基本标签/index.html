网页基本标签

1.标题标签 (header1 ~ header6)
2.段落标签 (paragraph)
3.换行标签 (break)
4.水平线标签 (horizontal)
5.字体样式标签
  • 加粗 : strong
  • 斜体 : em (emphasis)
6.注释和特殊符号

  • 注释 :

  • 空格 : &nbsp; (non breaking space)

  • 大于 : &gt; (greater than)

  • 小于 : &lt; (less than)

  • 版权 : &copy; (copyright)

7.图像标签

src : 图片路径

alt : 图片不显示时显示的文字信息

关于 ../ 和 ./

../ : 上一级目录, 父目录

./ : 当前目录

8.超链接

href : 链接地址

target : 链接窗口在哪里打开

  • _blank : 在新窗口打开
  • _self : 在当前窗口打开
9.锚点链接

使用a标签加name属性

name : 设置锚点名字

1.使用链接跳转到指定锚点

href : 填写需要跳转到的锚点名字, #后跟锚点名字

2.使用链接跳转到指定页面的指定锚点

10.功能性链接

邮件链接

在href属性中使用mailto:后跟具体邮箱

QQ推广

11关于块元素和行内元素

块元素 : 无论内容多少, 独占一行

行内元素 : 由内容大小决定宽度多少, 左右都是行内元素的可以排在一行

12.列表

1.有序列表

2.无序列表

3.自定义列表

dl : 标签

dt : 列表名

dd : 列表内容

13.表格标签

table : 标签

tr : 行

td : 列

  • colspan : 跨列属性
  • rowspan : 跨行属性
14.媒体元素

src : 文件路径

controls : 控件

autoplay : 自动播放

loop : 循环播放

视频

音频

15.iframe内联框架

内联框架就是在一个网页中嵌入另一个网页

src : 默认内联框架中显示的网页

frameborder : 框架是否显示边框(1 或 0)

width : 设置框架宽度

height : 设置框架高度

name : 设置框架的名字, 可用于设置a标签的target属性, 使a标签跳转的网页在内联框架中显示

16.表单

action : 表单数据提交到什么地方

method : 提交数据的方式

  • get : 我们可以在提交的URL中看到我们提交的信息, 不安全, 但是高效
  • post : 看不到我们提交的信息, 但是安全, 可以传输大文件

input : input标签

type : input标签根据标签属性type的值不同可以有不同的展现形式

  • text : 文本类型输入框
  • password : 密码类型输入框
  • submit : 提交类型输入框

name : 每个input标签的标识

17.单选按钮

设置input标签的type属性为radio

checked : 默认选中

注意 : 单选按钮存在组的概念, 也就是多个type为radio的单选按钮他们的name属性值需要一样, 这样才能实现只能选一个, 不然可以选多个

18.多选按钮

checked : 默认选中

设置input标签的type属性为checkbox, 同为一个按钮组的多选按钮需要设置name属性值一样

19.其他按钮

当input标签的type属性值为以下内容时

  • button : 普通按钮
  • submit : 提交按钮(提交表单数据)
  • reset : 重置按钮(重置表单数据)
  • image : 图片按钮(图片是一个按钮)
20.下拉框

name : 设置下拉框的名字

value : 设置下拉框的子元素值

selected : 默认选择

21.文本域

name : 文本域的名字

id : 文本域的id

cols : 文本域的宽度(列的数量)

rows : 文本域的高度(行的数量)

22.文件域

设置input标签的type属性值为file时, input标签将展现为文件上传按钮样式

name : 设置上传文件的名字(这个不是文件的真实名字, 是我们自己定义的)

23.input标签简单验证

  • 邮箱地址验证

    设置input标签的type属性值为email时, 当点击提交按钮, 将会自动验证输入框中的内容是否符合邮箱地址

  • URL验证

    设置input标签的type属性值为url时, 当点击提交按钮, 将会自动验证输入框中的内容是否符合url

  • 数字

    设置input标签的type属性值为number时, 输入框中只能输入数字, 不能输入其他内容

    name : 当前input标签的名字

    value : 默认值

    max : 最大值

    min : 最小值

    step : 步长

24.滑块

设置input标签的type属性值为range时, input标签展现形式为滑块

name : 当前input标签的名字

value : 默认值

max : 最大值

min : 最小值

step : 步长

25.搜索框

设置input标签的type属性值为search时, input标签展现形式为搜索框

name : 当前input标签的名字

placeholder : 输入框底部显示的内容

26.input标签的其他属性
  • hidden : 隐藏域
  • readonly : 只读
  • disable : 禁用
  • required : 内容不能为空
27.label标签增强鼠标可用性

for : label标签属性, 属性值为input标签的id值, 当点击label标签时, 将自动定位到for指定id的input标签

28.正则表达式验证

当要自定义验证时, 可以使用pattern属性, 属性值为一个正则表达式, 用于处理自定义验证

常用正则表达式 : https://www.jb51.net/tools/regex.htm
